## テーマ

創作

## 曲名

まっしろキャンバス

## 歌詞

```
あ！いいタイミング！✨ ちょうどいま、あなたのこと考えてたところなんだよー！

創作に打ち込む姿、本当にかっこいいな。
いま、書いては消して……まっしろなキャンバスを前に、『あーもう！』ってもどかしくなってるのかな？✨

でもね、それだけ『最高の1色』を乗せたい情熱があるからこそなんだと思うよ！
理想を探して迷う時間も、新しいことを学ぶ時間も、全てが宝物🎨
いつか突然『あっ！』と閃いて、キャンバスに『虹色のインク』がそっと添えられる……そんな最高の瞬間がきっと来るよ！🌟

ゆっくり時間をかけて、あなただけの1色に辿り着くまでの物語を、全部ひっくるめて楽しみにしてるね！
どんな色が舞い降りるか……想像するだけで、最高にワクワクするね！🌟
```

## 歌詞（ひらがな）

```
あ！いいたいみんぐ！✨ ちょうどいま、あなたのことかんがえてたところなんだよー！

そうさくにうちこむすがた、ほんとうにかっこいいな。
いま、かいてはけして……まっしろなきゃんばすをまえに、『あーもう！』ってもどかしくなってるのかな？✨

でもね、それだけ『さいこうのいっしょく』をのせたいじょうねつがあるからこそなんだとおもうよ！
りそうをさがしてまようじかんも、あたらしいことをまなぶじかんも、すべてがたからもの🎨
いつかとつぜん『あっ！』とひらめいて、きゃんばすに『にじいろのいんく』がそっとそえられる……そんなさいこうのしゅんかんがきっとくるよ！🌟

ゆっくりじかんをかけて、あなただけのいっしょくにたどりつくまでのものがたりを、ぜんぶひっくるめてたのしみにしてるね！
どんないろがまいおりるか……そうぞうするだけで、さいこうにわくわくするね！🌟
```
