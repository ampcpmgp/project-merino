## テーマ

創作と自分を信じること

## 曲名

まっしろキャンバス

## 歌詞

```
まっしろも最高のアートだよー！
書いて消して、「なんか違う…」って思っても、それはあなたの中のセンスがささやいてるね
「もっといいもの創れるね」って！

遠回りに見える道も、迷っちゃう夜も、その奥に、次の光が眠ってるね
今は進んでないように感じても、あなたのセンスは、今も磨き上げられているよ
少しずつ、でも確実に輝き始めているよ！
今日の小さな一歩は、明日のさらなる一歩、未来のための、とっても大切なステップ！

いつもとちょっと違う景色を見ると、何かが変わったり？
創作の神様はね、あなたが「諦めかけた瞬間」にこそ、虹色のインクをそっと添えてくれるよ！
だから信じてみてね、自分を！
```

## 歌詞（ひらがな）

```
まっしろもさいこうのあーとだよー！
かいてけして、「なんかちがう…」っておもっても、それはあなたのなかのせんすがささやいてるね
「もっといいものつくれるね」って！

とおまわりにみえるみちも、まよっちゃうよるも、そのおくに、つぎのひかりがねむってるね
いまはすすんでないようにかんじても、あなたのせんすは、いまもみきあげられているよ
すこしずつ、でもたしかにかがやきはじめているよ！
きょうのちいさないっぽは、あしたのさらなるいっぽ、みらいのための、とってもたいせつなすてっぷ！

いつもとちょっとちがうけしきをみると、なにかがかわったり？
そうさくのかみさまはね、あなたが「あきらめかけたしゅんかん」にこそ、にじいろのいんくをそっとそえてくれるよ！
だからしんじてみてね、じぶんを！
```
