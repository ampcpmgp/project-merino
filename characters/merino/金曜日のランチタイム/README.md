## テーマ

一週間の仕事を頑張って、だんだんと疲れてきた

## 曲名

金曜日のランチタイム

## 歌詞

```
お疲れさま～！
金曜日のランチタイム、頑張った自分にちょっと拍手しよ？
ここまで来れたんだから、もう十分すごいよ！
「あー、もうクタクタ…」って思うかもしれないけど、
それはあなたが一生懸命やった証拠。
疲れたって、全然OK！

週末はもう目の前だから、あとはちょっとだけの我慢。
終わったら、好きなこといっぱいしよ？
美味しいもの食べて、ゆっくりお風呂入って、
好きな音楽聴きながらゴロゴロするもよし！
「私、今日もよくやったな～」って自分を褒めて、甘やかしてあげてね。

あなたはもう十分頑張ってるから、無理しなくていいんだよ。
ゆっくり休んで、自分のペースで進んでいけば大丈夫！
いつも頑張ってるあなたに、たくさんの優しさを送ります～💕

今日も、そしてこれからも、
あなたらしくいられることを願ってるよ！
```

```
おつかれさま～！
きんようびのらんちたいむ、がんばったじぶんにちょっとはくしゅしよ？
ここまでこれたんだから、もうじゅうぶんすごいいよ！
「あー、もうくたくた…」っておもうかもしれないけど、
それはあなたがいっしょうけんめいやったしょうこ。
つかれたって、ぜんぜんおーけー！

しゅうまつはもうめのまえだから、あとはちょっとだけのがまん。
おわったら、すきなこといっぱいしよ？
おいしいものたべて、ゆっくりおふろはいって、
すきなおんがくききながらごろごろするもよし！
「わたし、きょうもよくやったな～」ってじぶんをほめて、あまやかしてあげてね。

あなたはもうじゅうぶんがんばってるから、むりしなくていいんだよ。
ゆっくりやすんで、じぶんのぺーすですすんでいけばだいじょうぶ！
いつもがんばってるあなたに、たくさんのやさしさをおくります～

きょうも、そしてこれからも、
あなたらしくいられることをねがってるよ！
```
