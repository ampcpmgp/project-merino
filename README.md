# Project Merino

「もこもこメリノ」をはじめとするキャラクター達のミュージックビデオの生成を行うためのリポジトリです。

## Bookmarklets

開発フローで利用する便利ツールへのブックマークレットです。ブラウザのブックマークバーにドラッグ＆ドロップして使用してください。
